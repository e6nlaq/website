# AJLシミュレーター 実装計画書 (v3)

## 1. 概要
現在のAJL合計スコアと、スコア対象となっている最低パフォーマンスを入力することで、次回のパフォーマンスに応じたスコアの推移をグラフ化します。また、パフォーマンスの範囲に応じた背景色を表示し、目標達成への目安を視覚的にサポートします。

## 2. 仕様
### 入力項目
- **現在の合計スコア:** 数値入力（現在のAJL合計スコア）
- **対象内の最低パフォーマンス:** 数値入力（任意）
    - ※ すでにD個（Algo:6, Heuristic:4）以上のコンテストに参加している場合、その中で最も低いperfを入力します。
    - ※ **空欄の場合は参加数がD個未満**とみなし、単純加算でシミュレーションします。
- **グラフの範囲:** 次回 perf の最小値と最大値（デフォルト: 0 〜 4000）

### 計算ロジック
次回のコンテストのパフォーマンスを $x$ としたとき：
- **最低perfが未入力の場合（追加枠がある場合）:**
    - 新スコア = `currentScore + 1000 * 2^(x/400)`
- **最低perfが $P$ の場合（入れ替えが発生する場合）:**
    - 新スコア = `currentScore + max(0, 1000 * 2^(x/400) - 1000 * 2^(P/400))`

### 表示内容・グラフ演出
- **シミュレーショングラフ:** 横軸 $x$ (perf), 縦軸 $y$ (score) の折れ線グラフ
- **背景色 (ReferenceArea):** 横軸の範囲に応じて、AtCoderのレート色で背景を塗り分けます。
    - 0-399: 灰, 400-799: 茶, 800-1199: 緑, 1200-1599: 水, 1600-1999: 青, 2000-2399: 黄, 2400-2799: 橙, 2800-: 赤
- **分析・補助線:**
    - 現在のスコアを水平線で表示。
    - 最低perfが入力されている場合、その位置に垂直線を表示。

## 3. 技術選定
- **グラフ:** `recharts`
- **UI:** `ToolCard`, `react-hook-form`, `zod`
